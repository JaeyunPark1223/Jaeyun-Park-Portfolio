# Jaeyun P Portfolio

This is a simple website for photos and artwork. You can edit it directly by changing the files.

## How to edit the website

1. Open this folder in Visual Studio Code.
2. Edit `index.html` to change your name, writing, artwork titles, captions, email, and Instagram.
3. Edit `style.css` to change colors, spacing, fonts, and layout.
4. Put your photo and artwork images in the `images` folder.
5. In `index.html`, replace image names like `images/artwork-1-placeholder.svg` with your real image file names.

Example:

```html
<img src="images/my-painting.jpg" alt="My painting title">
```

## How to publish

1. In GitHub Desktop, write a short summary like `Add portfolio website`.
2. Click **Commit to main**.
3. Click **Publish repository** or **Push origin**.
4. On GitHub.com, open the repository.
5. Go to **Settings** then **Pages**.
6. Choose **Deploy from a branch**.
7. Choose branch **main** and folder **/ root**.
8. Save.

GitHub will show your website link after it finishes publishing.
